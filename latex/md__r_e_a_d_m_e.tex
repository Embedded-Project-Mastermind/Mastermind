\item Present the purpose of the project and its main functionalities~\newline
 The purpose of the project is to develop a simple version of th egame Mastermind. It is a simple and funny game where the user has to guess a sequence of colors within a certain number of attempts. Some settings for the game can be chosen, like the dimension of the sequence to guess, the difficulty, the number of tentatives and the precence of doubles.
